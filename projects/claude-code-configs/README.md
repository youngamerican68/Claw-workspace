# Claude Code & Cursor Reliability Templates

> Battle-tested configurations for AI coding tools. The "dotfiles" movement for Claude Code and Cursor.

## Problem

Claude Code and Cursor are powerful but inconsistent. Developers waste hours figuring out:
- Which system prompts work best
- How to configure .cursorrules files
- What CLAUDE.md templates to use
- Per-framework settings that actually work

## Solution

A curated, tested library of configurations — optimized by framework, with community ratings.

## Structure

```
configs/
├── claude-code/
│   ├── system-prompts/
│   │   ├── general.md
│   │   ├── react.md
│   │   ├── python.md
│   │   └── rails.md
│   └── CLAUDE.md/
│       ├── react-template.md
│       ├── python-template.md
│       └── rails-template.md
├── cursor/
│   ├── .cursorrules/
│   │   ├── general
│   │   ├── react
│   │   ├── python
│   │   └── rails
│   └── settings/
└── ratings.json
```
