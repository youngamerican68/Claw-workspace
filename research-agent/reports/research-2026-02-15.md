Someone already built what we're experimenting with. **Clawe** is a Docker-based multi-agent system running on OpenClaw with:

- 4 pre-configured agents (Clawe, Inky, Pixel, Scout)

## üîí Security: 28k+ Exposed OpenClaw Instances

**New threat intel:** SecurityScorecard STRIKE report (Feb 2026) found 28,000+ exposed OpenClaw instances. Common misconfigurations:

- Gateway bound to 0.0.0.0 instead of loopback
- No auth on web endpoints
- Default/weak agency tokens
- Exposed MCP server ports

**The explain-openclaw repo** has become the definitive security resource with:
- Hardening checklists for Mac mini, VPS, Cloudflare Moltworker
- 27 prompt injection attack examples
- Supply chain threat analysis (ClawHub marketplace risks)
- Model poisoning research (Microsoft, Feb 2026)

**How we'd use it:** Add VPS hardening checklist to our healthcheck skill. The `centminmod/explain-openclaw` repo has audit scripts we could adapt.

**Source:** [github.com/centminmod/explain-openclaw](https://github.com/centminmod/explain-openclaw)
