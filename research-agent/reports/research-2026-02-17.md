## Security

### OpenClaw Security Sprint
The 2026.2.15 release includes significant hardening:
- **SHA-256 Migration**: Replaced deprecated SHA-1 for sandbox configuration hashing.
- **Secret Redaction**: Telegram bot tokens redacted from error messages and stack traces.
- **Sandbox Lockdown**: Blocked dangerous Docker configs (bind mounts, host networking, unconfined seccomp/apparmor).
- **XSS Prevention**: Control UI now serves bootstrap config as JSON with `script-src 'self'` CSP.
- **Path Sanitization**: Workspace paths sanitized before embedding into LLM prompts (prevents instruction injection via malicious directory names).
